Visualizing Git Concepts with D3 - The Basics

These are the most basic git commands for you to visualize.

git commit

Open this tutorial

We are going to skip instructing you on how to add/stage your files for commit in this explanation. Let's assume you already know how to do that. If you don't, go read some other tutorials.

Pretend that you already have your files staged for commit and enter "git commit" as many times as you like in the terminal box.

git branch

Open this tutorial

Creating branches just creates a new tag pointing to the currently checked out commit.

Type git commit and git branch commands to your hearts desire until you understand this concept.

git checkout

Open this tutorial

git checkout has many uses, but the main one is to switch between branches.
For example, to switch from master branch to dev branch, I would type 'git checkout dev'. After that, if I do a git commit, notice where it goes. Try it.

In addition to checking out branches, you can also checkout individual commits. Try it. Type "git checkout 77oa0ph" and see what happens.

Type git commit, and git branch, and git checkout commands to your hearts desire until you understand this concept.
